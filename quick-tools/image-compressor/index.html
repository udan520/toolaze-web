Frequently Asked Questions

Which image formats are supported? +

Toolaze supports JPG, JPEG, PNG, WebP, and BMP formats. All compressed images are output as optimized JPEG files.

Is this tool really free? +

Yes! Toolaze is 100% free with no ads, no registration, and no hidden fees. All processing happens in your browser.

Can I compress more than 100 images? +

The maximum batch size is 100 files per session. For larger batches, simply process them in multiple sessions.

Are my images uploaded to a server? +

No! All image processing happens locally in your browser using JavaScript. Your images never leave your device, ensuring complete privacy.

What is the maximum file size I can compress? +

There's no theoretical file size limit. However, very large images may cause browser slowdown depending on your device's performance and available memory. We recommend processing large files in smaller batches for the best experience.

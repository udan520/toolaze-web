# Intro 板块规范

## 推荐布局

Picflow 风格：标题 + 图标行 + 描述段落 + 底部卡片

## 字段

### content

- **推荐范围**：150-250 个单词
- **最佳长度**：180-220 个单词
- **最小**：不少于 120 个单词
- **最大**：不超过 300 个单词
- 使用 `\n\n` 分隔段落
- 每段建议 60-100 个单词

### iconRow（可选）

- 推荐 3-4 个图标
- 类型：`privacy`、`speed`、`unlimited`、`batch`
- 格式：`{ "icon": "privacy", "label": "Privacy" }`

### bottomCards（可选）

- 推荐 3 张卡片（Engine, Limit, Privacy）
- 每卡：`icon`、`title`、`desc`
- desc 建议 10-20 个单词

## 内容要求

- 深入阐述工具的核心价值和优势
- 可包含用户痛点、解决方案、技术优势
- 第一段：核心价值（60-80 单词）
- 第二段：技术优势或使用场景（60-100 单词）
- 第三段（可选）：补充说明（40-60 单词）

# Hero 板块规范

## 字段

- **仅保留**：`h1`、`desc`
- **不显示**：`sub` 字段（即使 JSON 中有）

## h1 要求

- 简洁明了，只包含核心工具名称
- 示例：`"Unlimited HEIC to JPG Converter"`
- 必须包含修饰词（Free 或 Online），见 [metadata.md](metadata.md)

## desc 要求

- 长度：1-2 句话，约 15-25 个单词
- 必须包含：工具功能 + 免费 + 简单操作说明
- 格式示例：`"Convert HEIC to JPG online for free. Simply drop your HEIC images below to convert them to JPG in seconds."`

## 最佳实践

- desc 简洁有力，突出核心价值
- 避免冗长描述
- 让用户快速理解工具能做什么

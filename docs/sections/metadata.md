# Metadata 与 H1 规范

## Meta Title 公式

`[Action] [Input] to [Output] [Category] ([Benefit 1], [Benefit 2] & [Benefit 3]) - Toolaze`

- Category 必须包含 "Online"
- Benefits 第一个必须是 "Free"
- 示例：`Convert PNG to WebP Online (Free, Batch & No Ads) - Toolaze`

## Meta Description

- **前 100 字符**：How to + 3 步骤 + 必须包含 Free 或 Online
- **后 50 字符**：技术优势 + 安全/隐私 + 必须包含 Free
- 示例：`How to convert PNG to WebP online? 1. Upload PNG 2. Auto-convert 3. Download WebP. Free online converter. Shrink file size by 35% with zero quality loss. Secure & Private.`

## H1 策略

- **基础格式**：`[Input] to [Output] Converter` 或 `[Tool Name]`
- **增强格式**（推荐）：`Free [Tool Name]` 或 `[Tool Name] Online` 或 `Free [Tool Name] Online`
- **必须包含**：至少一个修饰词（Free 或 Online）
- **组合原则**：最多 2 个修饰词，避免堆砌

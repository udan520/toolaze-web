# Kling 3.0 功能介绍

> **文件位置**：`docs/specs/kling-3.md`  
> **用途**：Kling 3.0 工具的功能介绍，作为所有 Kling 3.0 相关 SEO 内容的基础参考

## 产品概述

Kling 3.0 是快手（Kuaishou）推出的 AI 视频生成模型，采用统一多模态架构，支持文本、图像、音频、视频的输入与输出。

## 支持的功能

### 支持的输入格式
- **文本 (Text)**：自然语言描述
- **图像 (Image)**：参考图
- **视频 (Video)**：参考视频
- **音频 (Audio)**：参考音频

### 核心功能
- **多模态输入**：文本转视频、图转视频、参考转视频、视频内编辑统一架构
- **原生 4K/2K 输出**：支持 4K 和 2K 分辨率
- **15 秒视频**：可自定义时长，支持多镜头（最多 6 个镜头）
- **原生音频生成**：支持英语、中文、日语、韩语、西班牙语及方言口音，多角色对话
- **写实风格**：人物表现力强，动态自然
- **文字保留**：准确保留或生成招牌、品牌 logo 等文字元素
- **多镜头叙事**：正反打对话、交叉剪辑等智能多镜头能力

### 技术实现
- Kuaishou Kling 3.0 模型
- 统一多模态架构
- 云端 API 调用

## 不支持的功能

- ❌ 完全本地/离线处理（需要网络连接）
- ❌ 100% 浏览器端处理（需要服务器 API）
- ❌ 3D 模型生成

## SEO 内容编写注意事项

1. **只能写支持的功能**：只写上述"支持的功能"列表中明确列出的功能
2. **不能写不支持的功能**：绝对不能提及"不支持的功能"列表中的任何内容
3. **技术描述准确**：Kling 3.0 是 API 调用工具，**不能**说 "100% private processing" 或 "local processing"
4. **强调优势**：可强调 4K 输出、多语言音频、多镜头、写实画质等
5. **与 Seedance 2.0 区分**：Kling 3.0 突出 4K、多语言音频、多镜头；Seedance 2.0 突出多模态输入数量

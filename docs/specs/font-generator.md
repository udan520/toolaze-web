# Font Generator 功能介绍

> **文件位置**：`docs/specs/font-generator.md`  
> **用途**：Font Generator 工具的功能介绍，作为所有 Font Generator 相关 SEO 内容的基础参考

## 支持的功能

### 支持的字体类型
- Cursive（手写体）
- Fancy（装饰体）
- Bold（粗体）
- Italic（斜体）
- Gothic（哥特体）
- Tattoo（纹身字体）
- Calligraphy（书法体）
- Old English（古英语体）
- 3D（立体字体）
- Script（脚本体）
- 以及其他多种字体风格

### 核心功能
- 在线字体生成（浏览器端处理）
- 实时预览
- 复制粘贴功能（支持多种格式）
- 多种字体风格选择
- 自定义文本输入
- 客户端处理（100% 本地处理，不上传服务器）
- 免费使用（无广告）

### 技术实现
- 浏览器原生 API
- CSS 字体渲染
- 客户端 JavaScript 处理
- 本地处理（无服务器上传）

## 不支持的功能

- ❌ 字体文件下载（.ttf, .otf 等）
- ❌ AI 字体生成
- ❌ 自定义字体上传
- ❌ 云端处理
- ❌ 服务器端处理

## SEO 内容编写注意事项

1. **只能写支持的功能**：只写上述"支持的功能"列表中明确列出的功能
2. **不能写不支持的功能**：绝对不能提及"不支持的功能"列表中的任何内容
3. **技术描述准确**：只提及"技术实现"中列出的技术，不能夸大或添加未使用的技术
4. **字体类型限制**：只能写支持的字体类型，不能写其他未支持的字体

# Image Compressor 功能介绍

> **文件位置**：`docs/specs/image-compressor.md`  
> **用途**：Image Compressor 工具的功能介绍，作为所有 Image Compressor 相关 SEO 内容的基础参考

## 支持的功能

### 支持的格式
- JPG/JPEG
- PNG
- WebP
- BMP

### 核心功能
- 图片压缩
- 精确大小控制（KB/MB）
- 批量处理（最多 100 张图片）
- 客户端处理（100% 本地处理）
- 质量保持

### 技术实现
- 浏览器原生 API
- Canvas API 压缩
- 客户端处理

## 不支持的功能

- ❌ Video 压缩
- ❌ AI 技术
- ❌ 云端处理
- ❌ 服务器端处理

## SEO 内容编写注意事项

1. **只能写支持的功能**：只写上述"支持的功能"列表中明确列出的功能
2. **不能写不支持的功能**：绝对不能提及"不支持的功能"列表中的任何内容
3. **技术描述准确**：只提及"技术实现"中列出的技术，不能夸大或添加未使用的技术
4. **格式限制**：只能写支持的格式，不能写其他格式

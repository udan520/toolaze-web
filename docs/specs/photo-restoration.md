# Photo Restoration 功能介绍

> **文件位置**：`docs/specs/photo-restoration.md`  
> **用途**：Photo Restoration 工具的功能介绍，作为该页面 SEO 内容编写依据

## 支持的功能

### 输入与输出
- 单张图片输入（1 张）
- 生成单张结果图（1 张）
- 默认输出分辨率：1K
- 默认比例：Auto

### 核心功能
- 老照片修复（去除划痕、灰尘、噪点）
- 清晰度与细节增强
- 固定执行修复 + 上色（无开关）

### 固定交互规则（来自 PRD）
- 模型：GPT Image 2
- 提示词：隐藏给用户（前端内置固定提示词）
- 提示词固定为「修复 + 上色」
- 不允许用户切换模型
- 不展示历史记录

## 不支持的功能

- ❌ 批量处理
- ❌ 自定义提示词输入
- ❌ 用户手动切换模型
- ❌ 多图输入
- ❌ 上色开关（Colorize toggle）

## SEO 内容编写注意事项

1. 只能写单图修复流程，不写批处理能力  
2. 只能写固定修复+上色流程，不写开关或高级参数编辑  
3. 不在面向用户文案中出现 API 术语  
4. 强调「免费在线」「无需注册」「浏览器使用」等 C 端表达

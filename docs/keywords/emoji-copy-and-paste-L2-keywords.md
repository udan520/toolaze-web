## 使用建议

1. **主关键词**：在 H1、Title、Meta Description 中各出现 1 次；在正文前 100 字内出现 1–2 次。
2. **长尾关键词**：在 Intro、Features、How to Use、FAQ、Scenarios 中自然分布，每词 1–2 次，总关键词密度约 1–2%。
3. **内部链接**：后续 L3 页（如 heart、crying、fire 等）可用 “heart emoji copy and paste” 等作为锚文本链回 L2。

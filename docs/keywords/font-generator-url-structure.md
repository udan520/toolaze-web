## 💡 SEO 优化建议

### 1. 关键词密度

- **H1**：使用精确匹配的关键词（如 "Cursive Font Generator"）
- **Title**：主关键词 + 品牌（如 "Cursive Font Generator - Free Online Tool | Toolaze"）
- **Meta Description**：包含主关键词 + 1-2 个长尾关键词
- **内容**：自然嵌入相关关键词，密度 1-2%

### 2. 内容差异化

每个 L3 页面必须：
- 100% 唯一的内容（不能复制其他页面）
- 针对该特定关键词的详细说明
- 独特的使用场景和示例
- 独特的 FAQ（至少 5 个问题）

### 3. 内部链接

- 每个 L3 页面链接到 3-5 个相关 L3 页面
- 所有页面都有回到 L2 的链接
- 使用关键词作为锚文本

### 4. 技术 SEO

- 所有页面包含 JSON-LD 结构化数据
- 使用 canonical URL
- 优化图片 alt 文本
- 确保页面加载速度 < 3 秒

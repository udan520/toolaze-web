## 竞品驱动的 Toolaze 决策

### 顶部功能

- 使用单图上传。
- 使用 Women、Men、Custom 三个 Tab，避免把男女发型混在一个长列表中。
- Women 展示 11 个图片模板，Men 展示 11 个图片模板。
- Women 和 Men 的模板提示词内置，不向用户显示 Prompt 输入框。
- Custom 不显示模板卡片，只提供自然语言发型描述输入框。
- 当前模板图片为统一男女模特的占位资产，后续真实模板图直接覆盖同名 WebP。
- 每个模板保留可选 `referenceImage` 字段。真实模板图完成后，可在用户照片之外作为第二张隐藏参考图参与生成。
- 右侧默认 16:9 before/after。
- 不增加模型选择器。

### 声明策略

允许：

- free online
- no signup
- no watermark
- instant access
- visual preview

禁止：

- automatic face shape or hairline detection
- exact haircut result
- unlimited
- identical
- 100% private
- guaranteed face preservation
- hidden provider/model names in visible copy

## Handoff

Functional Acceptance：

- `image-to-image`
- `maxUploadImages: 1`
- `gender-tab-preset-first-with-custom`
- Women 11 个模板、Men 11 个模板、Custom 自由描述
- preset prompts hidden
- optional hidden template reference image
- default clear front-facing portrait
- hero before/after

SEO Content：

- 主攻 `ai hairstyle changer`。
- 使用 men、women、bob、pixie、bangs、buzz cut 等具体任务词。
- Hair Color Changer 只做内链。

Design：

- 预设卡必须显示可辨识发型，不使用抽象颜色块。
- Women 使用同一女性模特，Men 使用同一男性模特，确保模板之间只比较发型。
- 所有重复图片使用 WebP。
- 不在图片中写英文标签，避免本地化残留。

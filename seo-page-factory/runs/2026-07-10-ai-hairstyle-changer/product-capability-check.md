# Product Capability Check

Status: pass_with_limits

## 已确认能力

- 使用共享图片生成/编辑组件。
- 支持单张参考图上传。
- 支持 image-to-image prompt 编辑。
- 支持生成结果预览、下载和再次生成。
- AI Tools 场景页可使用 free、online、no signup、no watermark、instant access。
- 场景页可隐藏底层模型品牌。

## 需要保守表述

- 身份、脸部、姿势、服装和背景只能写 `aims to keep` 或 `visually consistent`，不能保证不变。
- 发型预览是视觉概念，不是专业剪发模拟。
- 生成速度受网络、队列和图片复杂度影响。

## 禁止声明

- 自动检测脸型
- 自动检测发际线
- 分析肤色或发质
- 精准预测真实剪发效果
- 结果与理发后完全一致
- 无限使用
- 照片绝对不存储或自动删除
- 专业发型师建议
- 内部模型、供应商或 API 路由
